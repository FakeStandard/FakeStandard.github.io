【C#】取得電腦名稱、作業系統版本、記憶體及文化特性資訊

April 19, 2021 · Polar Bear

Start#

近期為了在客戶的應用程式中添加一些裝置資訊，並將裝置資訊記錄成 LOG，以利下次客戶有問題時，可以先讓客戶提供有用資訊加以判斷。

引用 Microsoft.VisualBasic.Devices 命名空間，使用該命名空間下的 Computer 類別來取得資訊，Computer 類別提供許多屬性來操作電腦元件，這篇紀錄如何取得裝置資訊。

Computer computer = new Computer();

// 取得裝置名稱
Console.WriteLine(computer.Name);

// 實體記憶體總量
Console.WriteLine(computer.Info.TotalPhysicalMemory);

// 虛擬記憶體總量
Console.WriteLine(computer.Info.TotalVirtualMemory);

// 可用實體記憶體總量
Console.WriteLine(computer.Info.AvailablePhysicalMemory);

// 可用虛擬記憶體總量
Console.WriteLine(computer.Info.AvailableVirtualMemory);

// 完整作業系統名稱
Console.WriteLine(computer.Info.OSFullName);

// OS 平台識別項
Console.WriteLine(computer.Info.OSPlatform);

// OS 版本
Console.WriteLine(computer.Info.OSVersion);

// OS 文化特性
Console.WriteLine(computer.Info.InstalledUICulture);

除了使用 Computer 類別之外，也可以使用 ComputerInfo 取得裝置資訊

ComputerInfo info = new ComputerInfo();

上述看到 computer.Info.InstalledUICulture 文化特性，在該屬性下還有很多屬性可以使用，如下。

// 系統所安裝的 UI 文化特性
Console.WriteLine(computer.Info.InstalledUICulture);

// 預設行事曆
Console.WriteLine(computer.Info.InstalledUICulture.Calendar);

// 該文化特性的曆法集合
foreach (var calendar in computer.Info.InstalledUICulture.OptionalCalendars)
{
    Console.WriteLine(calendar);
}


// Culture 是否唯讀
Console.WriteLine(computer.Info.InstalledUICulture.IsReadOnly);

// 是否套用使用者選取的文化特性
Console.WriteLine(computer.Info.InstalledUICulture.UseUserOverride);

// 是否為中性文化特性
Console.WriteLine(computer.Info.InstalledUICulture.IsNeutralCulture);

// 數字、貨幣或百分比顯示格式
Console.WriteLine(computer.Info.InstalledUICulture.NumberFormat);

// 日期和時間的顯示格式
Console.WriteLine(computer.Info.InstalledUICulture.DateTimeFormat);

// 文化特性的書寫系統
Console.WriteLine(computer.Info.InstalledUICulture.TextInfo);

// ISO 639-1 兩個字母代碼
Console.WriteLine(computer.Info.InstalledUICulture.TwoLetterISOLanguageName);

// ISO 639-2 三個字母代碼
Console.WriteLine(computer.Info.InstalledUICulture.ThreeLetterISOLanguageName);

// 文化特性名稱
Console.WriteLine(computer.Info.InstalledUICulture.Name);

// 完整語系文化特性名稱
Console.WriteLine(computer.Info.InstalledUICulture.DisplayName);

// 英文文化特性名稱
Console.WriteLine(computer.Info.InstalledUICulture.EnglishName);

// 取得父文化特性
Console.WriteLine(computer.Info.InstalledUICulture.Parent);

// 語言和國家/地區組成之名稱
Console.WriteLine(computer.Info.InstalledUICulture.NativeName);

// 該文化特性如何比較字串
Console.WriteLine(computer.Info.InstalledUICulture.CompareInfo);

// 目前文化特性類型
Console.WriteLine(computer.Info.InstalledUICulture.CultureTypes);

// RFC 4646 標準識別
Console.WriteLine(computer.Info.InstalledUICulture.IetfLanguageTag);

// 輸入地區設定識別項
Console.WriteLine(computer.Info.InstalledUICulture.KeyboardLayoutId);

// 當前文化特性識別項
Console.WriteLine(computer.Info.InstalledUICulture.LCID);

// Windows API 定義之語言的三個字母代碼
Console.WriteLine(computer.Info.InstalledUICulture.ThreeLetterWindowsLanguageName);

Note#

有些屬性可直接印出結果，有些則是印出類名，這些類別下尚有許多屬性可使用，有興趣的可深入研究。

可使用類別

  • Microsoft.VisualBasic.Devices.Computer
  • Microsoft.VisualBasic.Devices.ComputerInfo
  • System.Globalization.CultureInfo
